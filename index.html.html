Projects

  • Weighted Ensemble Model for Tackling Fake News

    This project presents a robust approach to detecting fake news by leveraging ensemble learning with transformer-based text embeddings. The model utilizes BERT for contextual text representation and combines eight machine learning classifiers—Logistic Regression, SGD, XGBoost, SVM, Random Forest, AdaBoost, KNN, and Naive Bayes—through ensemble techniques like Majority Voting, Unweighted Averaging, and Weighted Averaging.

    Achieved 94.83% validation accuracy using Weighted Averaging across 8 classifiers.

    Published research paper

    Technologies Used: BERT (Bidirectional Encoder Representations from Transformers) Ensemble Learning Python Natural Language Processing Scikit-learn

  • Password-Based Security Lock System

    Embedded Systems Project using AVR Microcontroller Designed and implemented a secure electronic locking system using the ATmega16 microcontroller. Integrated a 4x4 matrix keypad for password input and a 16x2 LCD for real-time feedback. Developed an embedded C program to verify a 4-digit password and control a motorized lock mechanism based on input validation. The system loops for continuous operation and includes password re-entry and lock-close functionality to enhance user experience and security. Ideal for lockers, doors, or similar access-controlled environments.

    Key Highlights:

    Real-time password verification and display using an LCD Dynamic keypad scanning via GPIO port manipulation Motor control for automated locking/unlocking based on password logic Continuous loop for secure system reset and reuse

  • Domain Tagging and Label Generation

    Designed and implemented a system that analyzes textual content to identify domain relevance using natural language processing, keyword extraction, and graph-based analysis. The system processes input paragraphs, extracts significant keywords (using TF-IDF with bigram and trigram models), and retrieves related web pages via inlink-outlink analysis. It then constructs knowledge graphs to visualize content relationships and calculates domain-wise similarity percentages for tagging and content curation. This approach supports applications in SEO, academic research, and content moderation.

    Key features:

    Keyword extraction and similarity scoring using TF-IDF and fuzzy matching Graph-based domain tagging through inlink and outlink web page analysis Domain proportion prediction and label generation Modular pipeline for extensibility and scalable content analysis

    Technologies Used: Python Natural Language Processing(spaCy, TF-IDF, N-gram analysis) Beautiful Soup rapidfuzz Graph Construction Algorithms
